### Основные технологии:

React 
ReactDOM 
Redux 
Toolkit 
React Hook Form

### Стилизация:

Material-UI
Styled Components

### Скрипты:

Vite
TypeScript


## Запуск:

- Скопировать проект на компьютер
```
*git clone git@github.com:Sagaart-wold/frontend.git
```
- Перейти в папку проекта 
```
*cd frontend
```
- Установить зависимости
```
*npm i
```
- Запустить
```
*npm run build 
*npm run dev
```
